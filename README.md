# gdrive-telegram-bot
google drive control using telegram bot

Usage:
```shell
$ python gdrive_telegram_bot.py
```

require files
1. users.json

### users.json
```json
{
  "users": [
    {
      "first_name": "Wondo",
      "last_name": "Jung",
      "id": 12345678,
      "permission": "valid"
    }
  ]
}
```
